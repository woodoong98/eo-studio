### Develop Environment
1. node (node 8.12.0 버전)
2. npm (npm 6.4.1 버전)

### Use
1. front-default 디렉토리를 복사하여 프로젝트 명으로 변경한다.
2. 변경된 프로젝트 디렉토리로 들어온다.
3. 터미널에서 npm install을 실행하여 의존성 모듈을 설치한다.
4. gulp build-watch를 실행하여 초기 화면이 나오는지 확인한다. (기본 포트: 8888)
